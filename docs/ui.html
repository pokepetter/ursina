UI


Button


A clickable UI element with built-in highlight, pressed state, sound hooks, text handling and scaling. Button Documentation

Draggable


A UI element that can be clicked and dragged using the mouse.
Useful for movable windows, panels, or inventory items. Draggable Documentation

Tooltip


Displays text when hovering a UI element.
Can show descriptions, hints or help messages. Tooltip Documentation

Slider


A horizontal slider for selecting a float value between a minimum and maximum.
Comes with draggable handle and optional text label. Slider Documentation

ThinSlider


A minimal, compact slider variant with reduced visuals.
Takes less screen space than the regular Slider. ThinSlider Documentation

TextField


A simple field containing editable text.
Not auto-focused; meant for inline or small text modifications. TextField Documentation

InputField


A more advanced input widget that supports user text entry, caret movement, validation and events. InputField Documentation

ContentTypes


Used internally by InputField to categorize input types, such as text, numbers or custom sets. ContentTypes Documentation

Checkbox


A toggleable UI button representing a boolean state.
Can display checkmarks or custom icons. Checkbox Documentation

ButtonList


A vertical list of Buttons with automatic layout and optional spacing control.
Useful for menus, selections or tool panels. ButtonList Documentation

ButtonGroup


A group that ensures only one button is active at a time.
Ideal for radio-button-like selections. ButtonGroup Documentation

WindowPanel


A draggable, closable UI window with a title bar and customizable content area.
Suitable for tool panels, options windows, popup UIs. WindowPanel Documentation

FileBrowser


A UI browser for selecting existing files.
Shows directory contents visually and emits events on selection. FileBrowser Documentation

FileBrowserSave


A file browser specialized for saving operations, including filename entry. FileBrowserSave Documentation

DropdownMenu


A clickable menu that expands into a vertical list of selectable items.
Useful for compact settings controls or tool menus. DropdownMenu Documentation

RadialMenu


Displays buttons arranged in a circle around a central point.
Works well for quick-action menus or context radial selectors. RadialMenu Documentation

HealthBar


A horizontal bar representing health or any analog value.
Automatically updates scale based on a numeric input. HealthBar Documentation
